# Spaceship_Duel
2 player arcade game 

This was my first intermediate project using pygame!

Controls:
- Player 1: 'W' -> Up
            'S' -> Down
            'A' -> Left
            'D' -> Right
            'Left-Ctrl' -> Fire Bullet
            
       
- Player 2:
             'Up-Arrow' -> Up
             'Down-Arrow' -> Down
             'Left-Arrow' -> Left
             'Right-Arrow' -> Right
             'Right-Ctrl' -> Fire Bullet
