# Spaceship_Duel
2 player arcade game 

This was my first intermediate project using pygame!

Controls:
- Player 1: 'W' -> Move-Up
            'S' -> Move-Down
            'A' -> Move-Left
            'D' -> Move-Right
            'Left-Ctrl' -> Fire Bullet
            
       
- Player 2:
             'Up-Arrow' -> Up
             'Down-Arrow' -> Move-Down
             'Left-Arrow' -> Move-Left
             'Right-Arrow' -> Move-Right
             'Right-Ctrl' -> Move-Fire Bullet
